# Mongo Chat App

A simple chat application built with Node.js, Express, MongoDB, and EJS.

## Features
- Create, edit, and delete chat messages
- MongoDB as database
- Clean and simple UI

## Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- EJS templates

## Usage
1. Clone the repo
2. Run `npm install`
3. Run `node index.js` or `nodemon index.js`
4. Open `http://localhost:8080`
